# About Me

I'm a **Frontend Web Developer specializing in JavaScript, React, Next.js and modern web technologies**, currently expanding my skills toward becoming a **MERN Stack Developer**.

My journey into web development began in **December 2022**, when I purchased my first laptop after saving money while working abroad as a Sales Representative. I had dreamed of becoming a developer since 2019 and that experience helped me develop strong **discipline, adaptability and problem-solving skills**.

I have experience building **responsive and user-focused web applications** using **HTML, CSS, Bootstrap, Tailwind CSS, JavaScript, JQuery, WordPress, React and Next.js**. Currently, I'm expanding my backend development skills with **Node.js, Express.js and MongoDB** to become Full-Stack Developer.
